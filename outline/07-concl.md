# Section 7: Conclusion

## Flow
P1: Supercomputer as single machine
P2: ORCA realizes this vision
P3: Future: complete observability spectrum

## Paragraph Outlines

**P1: The Vision**
- Supercomputer = pretending to be single machine
- Homogenous architecture, all workers running same workload
- Connected by high-speed PCI-esque bus
- Compute on it as single machine

**P2: ORCA's Contribution**
- Observe it the same way: unified view via (swid, rank) indexing
- Control it the same way: cluster-scale programmability
- Three-stage evolution: offline → streaming → active

**P3: Towards Ideal Observability**
- Telemetry on tap, cost paid only when necessary
- Dynamic instrumentation + control semantics
- Complete feedback loop: operator ↔ application
